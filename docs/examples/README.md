# 範例程式碼

本目錄存放可執行的範例程式碼，包括：

- API 使用範例
- CLI 工具範例
- Script 範例
- Demo 專案

## 目錄結構

```
examples/
  ├── api/              # API 使用範例
  ├── scripts/          # 工具腳本
  └── demos/            # 完整 Demo 專案
```

## 範例分類

### API 範例
- TypeScript API 客戶端使用
- Supabase 客戶端範例
- 認證與授權範例

### Scripts
- 資料庫遷移腳本
- 測試資料產生器
- 備份與恢復腳本

### Demos
- 最小可執行專案
- 功能展示專案
