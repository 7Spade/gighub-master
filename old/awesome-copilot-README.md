# Awesome Copilot Collection

簡短說明
- **名稱**: Awesome Copilot
- **描述**: Meta prompts that help you discover and generate curated GitHub Copilot chat modes, collections, instructions, prompts, and agents.
- **標籤**: `github-copilot`, `discovery`, `meta`, `prompt-engineering`, `agents`

包含項目（節錄）
- `prompts/suggest-awesome-github-copilot-chatmodes.prompt.md` — prompt
- `prompts/suggest-awesome-github-copilot-collections.prompt.md` — prompt
- `prompts/suggest-awesome-github-copilot-instructions.prompt.md` — prompt
- `prompts/suggest-awesome-github-copilot-prompts.prompt.md` — prompt
- `prompts/suggest-awesome-github-copilot-agents.prompt.md` — prompt
- `agents/meta-agentic-project-scaffold.agent.md` — agent

如何使用
- 若專案或工具支援 `mcp_awesome-copil_load_collection`，可直接載入 collection：

  ```pwsh
  # 範例（示意）：
  # 請依實際環境呼叫相對應工具或 CLI
  mcp_awesome-copil_load_collection --id awesome-copilot
  ```

- 在本地瀏覽或引用這些 prompt/agent 檔案：
  - 路徑參考於 collection：`prompts/`、`agents/`

建議用途
- 快速產生或探索適合 GitHub Copilot 的 chat modes、instructions 或 agent 範本。
- 作為 prompt-engineering 或 Agent 開發的起始範例與集合。

貢獻
- 歡迎將新的 prompt、instruction 或 agent 加入 collection，並更新 `awesome-copilot.collection.yml`。
